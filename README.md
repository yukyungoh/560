# ADPROJECT 2 - 영화 관련 감성반응 분석 프로그램 제작 프로젝트
### w. 파이썬을 활용한 웹 크롤링 

![슬라이드2](https://user-images.githubusercontent.com/54924772/87314734-a095f000-c55e-11ea-9e77-0593d87f1efe.jpeg)
![슬라이드3](https://user-images.githubusercontent.com/54924772/87314738-a1c71d00-c55e-11ea-967b-915f2721c1e9.jpeg)

##### 광고홍보학 전공에서 다수의 팀 프로젝트를 진행할 때, 애용했던 데이터 분석 사이트가 있다.
##### 바로 검색어별로 트위터, 인스타그램, 페이스북, 네이버 등의 SNS 채널별로 검색량의 추이, 검색어 관련 감성 반응 등을 분석하여 결과 레포트를 제시해주는 '썸트렌드' 이다.
##### 나 뿐만 아니라, 대다수의 대학생들이 '썸트렌드' 사이트를 이용하여 브랜드, 혹은 특정 단어에 대한 감성 반응은 어떤지 유추하곤 한다.
##### 평소 이와 같은 데이터 처리, 데이터 분석에 관심이 많았던 터라, AD프로젝트의 주제를 '웹크롤링을 통한 감성 반응 분석 프로그램 제작' 으로 잡고, 이를 구현해보기로 했다.
###### '웹 크롤링'을 통해 감성 반응을 분석하기로 한 것은 파이썬 코드를 활용하면 생각보다 쉽게 웹 크롤링을 구현할 수 있고, 데이터 분석 분야에서 '웹 크롤링'이 폭넓게 활용되고 있었기 때문이다.

![슬라이드4](https://user-images.githubusercontent.com/54924772/87314743-a390e080-c55e-11ea-83ce-a0d7d788872a.jpeg)

##### 단순히 프로그램을 제작하는 것이 아니라, 특정한 상황을 가정한 후 프로그램을 제작하고 이를 발표하는 것이 더 흥미로울 것 같았다.
##### 때문에 나를 '영화사 집'의 홍보팀 직원으로 가정하고 해당 제작사 영화들의 흥행 정도를 홍보해야하는 업무를 맡은 것으로 가정했다.
##### 감성 반응 분석 대상을 '영화사 집이 제작한 영화들'로 좁힌 셈이다.

![슬라이드5](https://user-images.githubusercontent.com/54924772/87314758-a7246780-c55e-11ea-9356-b418296cb35c.jpeg)

##### 전공 수업을 수강하면서, 프로그램 제작 전에 '프로그램 요구사항'을 분석하고, 기술하는 단계가 중요하다고 배웠기 때문에 나름대로의 프로그램 제작시, 요구 사항들을 분석했다.
